    Welcome to Project Management made easy

    Basilisk is an easy to use, open source project management solution designed by programmers for progammers. Basilisk supports multiple projects as well as multiple repositories, even letting you create repositories as you need on the fly. Additionally, Basilisk features a easy to use, advanced bug tracker that allows you to easily cross reference reports to existing code. Have a project that uses large amounts of binary files that you want to keep out of version control? Basilisk can help you with that, as it proviedes it's own custom versioning support for these types of files that can be applied down to individual files. Basilisk will let you control every aspect of your projects all with an easy to use interface
